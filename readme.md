# JiLueHelper

This software is a tool for player of "SanGuoKill" to combine skills of Zuo You more conveniently.

## Table of Contents

- [Requirements](#requirements)
- [Install](#Install)

## Requirements

- numpy
- PyQt5
- pandas
- openpyxl

## Install

Use the following command to generate an executable file

```bash
pyinstaller -F -w main.py
```
